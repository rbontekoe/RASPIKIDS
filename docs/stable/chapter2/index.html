Wat heb je nodig

  • Een Raspberry Pi 3B+.
  • Een klaargemaakt micro SSD-kaartje voor de Raspberry Pi.
  • Een voeding voor de Raspberry Pi 3B+.
  • Een HDMI-monitor.
  • Een USB-toetsenbord.
